# jeanne-log

Next.js와 Tailwind CSS로 구축한 개인 기술 블로그입니다.

## 주요 기능

- **Next.js App Router**: 최신 Next.js의 라우팅 및 렌더링 기능을 사용합니다.
- **Tailwind CSS v4**: 최신 버전의 Tailwind CSS를 사용하여 모던하고 반응형 디자인을 제공합니다.
- **Markdown 지원**: `react-markdown`을 사용하여 마크다운 파일로 작성된 블로그 포스트를 렌더링합니다.
- **다크 모드**: 다크 모드를 기본적으로 지원합니다.
- **반응형 디자인**: 모든 기기 크기에 최적화되어 있습니다.


## 프로젝트 구조

- `app/`: Next.js App Router 페이지 및 레이아웃
- `components/`: 재사용 가능한 React 컴포넌트
- `services/`: 데이터 페칭 및 비즈니스 로직
- `contents/`: 블로그 포스트 마크다운 파일

## 기술 스택

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React](https://react.dev/)
